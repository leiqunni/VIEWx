dePic

概要

画像ビューアです。.NET を学び初めて早数年。作り直してみました。

ダウンロード

dePic release 1 [1.55MB] - 2012年01月17日
dePic release 1 source code [5.30MB] - 2012年01月17日 (Visual Basic 2010 Project)

スクリーン ショット

動作環境

.NET Framework 2.0 以上

インストール

適当な場所に展開してください。レジストリは使いませんが、下記場所に設定ファイルを作ります。
%LocalAppData%/DarkEra_Software/dePic.exe_Url_<hash>/<version>/user.config

アンインストール

フォルダごと削除して下さい。

使い方

キーボードの Space や Backspace や Enter や、 マウスのホイールでなんとかなります。画像の入ったフォルダごとドロップできます。
